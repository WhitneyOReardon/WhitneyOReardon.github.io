Week 10 | Electronics 2

This week I used Tinkercad to practice coding and to model circuits.


Videos

In the first video we practiced using the different components of Tinkercad circuits with an ardino, breadboard, LEDs, resistors, buttons, and potentiometers. The resistors and potentiometer help to reduce the current so that the LEDs don't short circuit, hence their name "current limiting resistors". Also, the dial on the potentiometer lets you the change the resistance for that current.


I accidentally wrote over the code that I used to follow along with the second video, but that lesson was about the basics of coding. We were shown how the create and manipulate variables using mathmatical and logical operators. Also, we learned how to print things to the serial monitor, which displays the results of whatever the arduino computes. In addition, we used if/else statements to create conditions for pieces of code to run.

In the last video we combined our knowledge of circuits and coding to create a simulation where an LED turned on only when a button was pressed. Here, we set the pinMode for the button as INPUT and the LED as OUTPUT. With this, the pin for the button could be read as having either a low or high electrical signal. In the code we arranged it so that if the button had a low signal, meaning the button was pressed, the LED would turn on then off.
